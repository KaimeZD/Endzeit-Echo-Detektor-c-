This is my first approach trying to visually and soundwise imitate the "Echo detector" from the Video game "Stalker 2".
The Code is C++ and it runs on the ESP32. I specifically tested it on the
"ELEGOO ESP32 Entwicklungsboard USB-C, 2.4GHz Dual Modus WiFi+Bluetooth Dual Core Mikrocontroller
CP2102 Chip"
The pin-layout can be seen in the "header" of the Code. Use the Arduino IDE to upload it to the ESP32.
Remember to use resistors for the levers.
The Components mostly run on 3,3V. It is recommended to have access to a 3D-printer for good results. You need
a good looking casing for the effects to really add up.
